Live website: https://smart-campus-assistant.vercel.app/

My Smart Campus Assistant is a student-focused web platform designed to simplify daily academic needs. It provides an easy Duty Leave Monitoring system for tracking attendance-related approvals, personalized Roadmaps to guide learning paths and skill development, and dedicated Placement Resources that help students prepare for internships and jobs with updated materials and insights. The platform brings everything into one convenient space, making campus life more organized and efficient.
